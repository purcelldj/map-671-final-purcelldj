Washington State Hazards Map

University of Kentucky Geography

I created this map for my final project for the Course MAP671. It depicts natural hazard areas in Washington State, particularly Geologic hazards such as Faults, Volcanoes, and Tsunamis. Volcano data shows how to appropriately respond to particular volcanic hazards, including lahar evacuation routes. Fault data shows the numerous area faults capable of causing local damaging earthquakes. Tsunami data shows tsunami hazard zones, evacuation routes, and assembly shelters.

This map was created using QGIS 3.40.10 and Mapbox, with data provided by the Washington State Department of Natural Resources Open Data Portal (https://geo.wa.gov/). This map was authored by Daniel Purcell for the University of Kentucky New Maps Plus program. Please view my github repository for this webpage at this link.
